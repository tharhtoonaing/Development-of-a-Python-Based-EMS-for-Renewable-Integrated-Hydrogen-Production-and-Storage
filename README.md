# Hydrogen EMS Python Project

This project starts with a hydrogen production calculator and will later be expanded into a Python-based energy management system for renewable hydrogen production.

## Current features
- Calculates hydrogen production from electrolyzer power, operating hours, and efficiency
- Uses HHV of hydrogen as the energy basis

## Next steps
- Add hourly solar and wind input
- Add hydrogen storage tank
- Add hydrogen demand
- Add battery storage
- Add plots with matplotlib
- Add CSV input with pandas
- Add machine learning with scikit-learn
- Add optimization with Pyomo
